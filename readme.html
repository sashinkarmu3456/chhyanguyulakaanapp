Summary to Setup & Configure Your Website

  1. The software is developed within the .NET platform, requiring users to have Microsoft .NET Framework 4.8 and Microsoft Visual Studio 2022 installed on their system.
  2. Installation of MSSQL Server 2019 or higher is necessary for the database.
  3. Windows Authentication should be enabled in SSMS
  4. The "AttachDBFilename" option is applicable to Microsoft SQL databases within the project's App_Data directory, located in the website's root folder. The connection string is stored in the web.config file, facilitating easy testing. Additionally, users can utilize an SQL script file within the project's db folder by copying and applying it through Microsoft SQL Server Management Studio.
  5. After setup, users can run the program either on an IIS server or directly through Microsoft Visual Studio 2022.
  6. Running from Microsoft Visual Studio 2022 involves importing the source file and accessing the default.aspx file.
  7. Upon opening the default.aspx file, users can click the Browser button to launch the website.
  8. The website will load and display the menu items.
  9. A default user 'admin@cyl.com' exists, with the password 'admin', granting administrative access upon logging in.
  10. Alternatively, users can log in with customer rights, provided they sign up or register first.
  11. Additional details are available in the accompanying snapshots.
